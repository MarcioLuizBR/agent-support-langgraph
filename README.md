# agent-support-langgraph
AI support agent using LangGraph with tool integration and stateful workflows. Agente de suporte com LangGraph, utilizando tools e fluxo baseado em estado

Mini projeto de agente com LangGraph.

## Funcionalidades
- Respostas gerais com LLM
- Uso de tool para consulta de status
- Fluxo com decisão (agent → tools → agent)

## Tecnologias
- LangGraph
- LangChain
- OpenAI
